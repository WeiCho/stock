# taiwan-stock skill

用自然語言查台股，整合技術面、籌碼面、回測、新聞基本面，並可輸出 TradingView Pine Script。

## 觸發時機

以下任何情況都應使用本 skill：

- 詢問個股分析：「分析台積電」、「2330 怎麼樣」、「幫我看一下聯發科」
- 技術面：「台積電技術面」、「2454 日K 均線」、「2330 週K」
- 籌碼面：「外資買了多少台積電」、「2330 三大法人」、「台積電籌碼」
- 回測：「2330 回測黃金交叉」、「台積電 KD 低檔交叉勝率」、「RSI 超賣回測」
- Pine Script：「幫我出台積電的 Pine Script」、「2330 ma_cross pine」
- 新聞/基本面：「台積電最新新聞」、「2330 本益比」、「台積電 EPS」
- 大盤：「今天大盤怎樣」、「加權指數」、「今日漲跌」
- 法人排行：「今天外資買超前五名」、「三大法人買超排行」
- 籌碼掃描：「找外資連買 3 天以上的股票」、「投信連買的股票」

## 使用方式

確認 server 已啟動（`localhost:8000`），然後執行：

```bash
cd ~/.claude/skills/taiwan-stock
source .venv/bin/activate
python main.py "<使用者查詢>"
```

輸出為 JSON，由 Claude 解讀後以中文回覆使用者。

## 注意

- 第一次查詢某支股票時會自動下載 10 年歷史（約 5–15 秒），請告知使用者稍候
- 若 server 未啟動，main.py 會印出啟動指令
- 資料僅供個人研究，不構成投資建議
